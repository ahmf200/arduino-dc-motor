# arduino-dc-motor
Controlling a DC motor with an infrared receiver.

## Equipment used:
Arduino Uno
1x DC motor
1x L293D chip
1x IR receiver with remote
1x power supply module

## Remote button map:
Using irrecv_remote.decodedIRData.command

69 = Power button
12 = Button 1
24 = Button 2
94 = Button 3
